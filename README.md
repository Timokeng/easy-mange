问题清单

1，技术栈以HTML+CSS+JS为主，捎带基础Vue框架，非单页网页应用，代码量较大，结构不佳，加载速度成问题，并且存在一些现实细节上的问题。
2，跳转页面通过window.location完成，过程过于简单粗暴，加载速度成问题。
3，图片上传内容未完成。
4，设计一些页面传值通过在url值添加内容传递，存在安全隐患。
5，基本未使用缓存技术，每次内容都通过后台获取，服务器压力可能较大。
